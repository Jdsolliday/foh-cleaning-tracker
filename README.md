# FOH Cleaning Task Tracker

A web-based task management system designed for front-of-house restaurant operations. The app tracks cleaning tasks, assigns responsibility, and automatically schedules recurring work.

## Features
- Add, edit, and delete tasks
- Mark tasks complete with automatic rescheduling
- Dashboard showing task status (Overdue, Due Soon, On Track)
- Persistent cloud database (Supabase)
- Real-time updates across users

## Tech Stack
- Python
- Streamlit
- Supabase (PostgreSQL)
- Pandas
